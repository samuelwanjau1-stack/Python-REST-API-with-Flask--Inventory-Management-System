# Inventory Management System (Flask REST API)

A robust backend inventory management system built with Flask, featuring full CRUD operations and integration with the OpenFoodFacts API.

## Project Structure


## Features
* **Flask RESTful API:** Manage inventory using standard HTTP verbs.
* **External API Integration:** Query real-time product data via OpenFoodFacts.
* **Comprehensive Testing:** Fully tested endpoints using `pytest`.
* **CLI Support:** Built-in command line interaction.
